# Button

Use buttons to trigger actions, drive direction, accomplish tasks or to link.

#### Writing Labels

Button labels use Title Case and clearly explain what happens when the button is activated. Keep labels as short as possible.

#### Button order

Order buttons primary action on a page or modal, the highest priority to the right.
Buttons assist user to complete tasks confidently and to feel secure about the next action they are taking.

## Button variants

### Solid buttons

**Solid buttons** are used to drive user action, there is only one primary action button per page. Solid buttons display filled with a colour and inverted white text eg. Blue, Green commonly used for:

- Sign off,
- Save,
- Apply,
- Approve etc.

Modals and forms use the primary action header. Not every screen requires a primary button. _Never use a primary red action button_.

```jsx live=true
<div className="grid grid-small">
  <Button color="primary">Primary</Button>
  <Button color="default">Default</Button>
  <Button color="success">Success</Button>
  <Button color="danger">Danger</Button>
</div>
```

### Inverse buttons

**Inverse button** are used as second primary actions alongside a primary action. Use grey a bordered button and text eg. Blue, Green, Red most commonly use for:

- Reject
- Cancel Campaign
- Save

**Inverted red buttons** are used as second primary actions alongside a primary action these are used delete, reject, cancel campaign.

```jsx live=true
<div className="grid grid-small">
  <Button color="primary" variant="inverse">
    Primary
  </Button>
  <Button color="default" variant="inverse">
    Default
  </Button>
  <Button color="success" variant="inverse">
    Success
  </Button>
  <Button color="danger" variant="inverse">
    Danger
  </Button>
</div>
```

### Borderless buttons

**Borderless button** is used for lower priority actions and when visually more suited than an inverse button.
borderless buttons have no border until hovered.

```jsx live=true
<div className="grid grid-small">
  <Button color="primary" variant="borderless">
    Primary
  </Button>
  <Button color="default" variant="borderless">
    Default
  </Button>
  <Button color="success" variant="borderless">
    Success
  </Button>
  <Button color="danger" variant="borderless">
    Danger
  </Button>
</div>
```

### Link buttons

**Link button** is a special case button used for inline actions that should retain the text flow, similar to anchor link.

```jsx live=true
<>
  <h4>
    This is an <Button variant="link">inline link button</Button> in an h4 tag.
  </h4>
  <p>
    This is an <Button variant="link">inline link button</Button> in a p tag.
  </p>
</>
```

## Button sizes

### Large buttons

**Large button** is designed to have larger text and padding.

```jsx live=true
<div className="grid grid-small">
  <Button color="primary" size="large">
    Primary
  </Button>
  <Button color="default" size="large">
    Default
  </Button>
  <Button color="success" size="large">
    Success
  </Button>
  <Button color="danger" size="large">
    Danger
  </Button>
</div>
```

## Full width buttons

**Full width** creates a 100% width button, sizing multiple buttons evenly when in a flex parent container.

```jsx live=true
<>
  <div className="grid grid-small">
    <Button size="large" fullWidth color="primary">
      Primary
    </Button>
    <Button size="large" fullWidth>
      Default
    </Button>
  </div>
</>
```

## Button states

**Loading** and **disabled** states can be applied to buttons.

```jsx live=true
<>
  <div className="grid grid-small">
    <Button isLoading color="primary">
      Loading
    </Button>
    <Button disabled>Disabled</Button>
  </div>
</>
```

## Icon buttons

default buttons support an icon size no larger than `16px`.
large buttons support an icon size no larger than `20px`.

### Icon

Create an icon-only button

```jsx live=true
<div className="grid grid-small">
  <Button aria-label="search" color="primary" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />} />
  <Button aria-label="search" color="default" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />} />
  <Button aria-label="search" color="success" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />} />
  <Button aria-label="search" color="danger" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />} />
</div>
```

### Round

When using an icon only, the button can be round.
Round icon Buttons have a special loading state, where the spinner aligns to the button outline, and the icon remains visible.

```jsx live=true
<div className="grid grid-small">
  <Button aria-label="search" round color="primary" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />} />
  <Button aria-label="search" round color="default" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />} />
  <Button aria-label="search" round color="success" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />} />
  <Button aria-label="search" round color="danger" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />} />
  <Button
    aria-label="search"
    isLoading
    round
    color="danger"
    icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />}
  />
</div>
```

### Icon with label

When combined with a label, the icon renders before the label

```jsx live=true
<div className="grid grid-small">
  <Button color="primary" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />}>
    Primary
  </Button>
  <Button color="default" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />}>
    Default
  </Button>
  <Button color="success" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />}>
    Success
  </Button>
  <Button color="danger" icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />}>
    Danger
  </Button>
</div>
```

### Describing icon buttons with a popover

Wrap in a Popover to indicate the button's purpose

```jsx live=true
<div className="grid grid-small">
  <Popover popoverContent="Search recent posts" theme="dark" placement="top" id="search-popover">
    <Button aria-label="Search recent posts" round icon={<SvgSymbol href="./assets/svg-symbols.svg#search" />} />
  </Popover>
</div>
```
