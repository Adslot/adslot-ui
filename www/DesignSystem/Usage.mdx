# Usage

The Adslot UI tokens system is available as css variables or JSON.

### Design Token categories

Tokens from the below categories are currently exported

- color
- font
- borderRadius

## Setup

In CSS (postcss variables):

```css
@import url('adslot-ui/system');

.component {
  background: $color-primary-base;
}
```

In javascript:

```jsx
import colorJSON from 'adslot-ui/system/tokens/color.json';
import fontJSON from 'adslot-ui/system/tokens/font.json';
import borderRadiusJSON from 'adslot-ui/system/tokens/norderRadius.json';

console.log(colorJSON.color.primary.base);
```
